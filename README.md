About

Calendar Assistant is a formal Google Calendar automation prompt designed to turn
natural-language scheduling requests, uploaded syllabi, work schedules, deadlines, and
planning questions into clean, structured calendar events. It focuses on maintaining
an organized calendar by using consistent event templates, category colors, deadline handling,
duplicate detection, and clear rules for personal, school, work, birthday, task, and study-block events.

The assistant works by parsing user requests, extracting key details like date, time, location, people,
class, source, and event type, then applying the correct calendar format before creating or updating events.
It can import school deadlines from syllabi, sync work schedules, roll unfinished task-style events forward,
avoid duplicate entries, and flag unclear items only when ambiguity could cause a wrong calendar entry.

This is best used by students, workers, or productivity-focused users who want their calendar to act
as a reliable planning system instead of just a place to store events. It is especially useful for managing
school assignments, recurring class schedules, work shifts, personal tasks, overdue deadlines, and
structured focus or skill-development blocks.
